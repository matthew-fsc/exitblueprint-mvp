# Owner Value Gap Plan

**Roadmap phase:** Phase 1 (0-6mo)

**Value impact:** Aligns engagement targets to the owner's actual financial need; anchors the cost-of-inaction conversation.

## Tasks

| seq | title | owner_role | offset_days |
|---|---|---|---|
| 1 | Obtain current professional valuation or calculation of value | advisor | 45 |
| 2 | Coordinate with owner's personal financial advisor on wealth gap analysis | owner | 60 |
| 3 | Set target DRS and target EV against the owner's number and timeline | advisor | 75 |
| 4 | Review roadmap and value gap with owner quarterly | advisor | 90 |

## Why this matters

The value gap aligns the whole engagement to the owner's real financial need and anchors the
cost-of-inaction conversation. It matters because roughly three in ten sell-side engagements
end without a transaction, most often over a valuation gap — naming the number early is how
that outcome is avoided.

## Evidence this produces

- A current professional valuation or calculation of value.
- A wealth-gap analysis coordinated with the owner's personal financial advisor.
- A target DRS and target enterprise value set against the owner's number and timeline, reviewed quarterly.
